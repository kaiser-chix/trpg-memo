# 第8話『帰郷』

## ◉ 導入
- 終戦後、領主たちや、「ブルーバレット」（ミスティ、ヘレン）や、我々「黄昏」は戦後処理に明け暮れていた
  - 我々「黄昏」は、領民からは、領主お抱えのパーティとして認識されている
- 3 ヶ月ほど経ち、やっと戦後処理の目処がついてきたところだ
- 今は領主からの依頼を済ませ、領主の館で一息をついているところだ
  - 領主の館の休憩室の中の一室は、「黄昏」専用の部屋となっていた
  - ドルチェはゴロゴロしている
- 休憩室の扉をノックされた
  - ドルチェは起き上がり、迎えに出た
  - ミスティだ
    - ドルチェはミスティをナンパしている
    - パーティメンバーはもう止めるのを随分前から止めたようだが、呆れた目でそれを見ている
  - ミスティは長い休暇を言い渡されたようだ
    - 本人としては、まだ働ける、と不満そうなところもある
    - 友人としてのお願いとして、里帰りに付き合って欲しいらしい
      - 領主様次第だけど、問題はないと返した
    - ミスティの故郷は、グランガチが出た湖の更に北、山を 3 つほど越えた辺りにあるらしい
    - 故郷は竜の襲撃で滅び、生き残りは数えるほどしか居ないそうだ
    - クラウディアやミスティの身内も亡くなり、クラウディアはその一件から狂ってしまったらしい
    - その後に生き残りは散り散りになり、今まで故郷に帰ることもできなかったそうだ
    - 20 年ほど前の出来事であるため、竜はもう居座っていないとは思うが、念の為保険として「黄昏」に付いてきて欲しいらしい
  - そこで、領主ドルチェが部屋に入ってきて、面白そうだから、と同行を申し出た
    - 領は大丈夫なのだろうか……
      - 落ち着いてきたし、いいんじゃない？　とのこと
