# その灯の名は　キャラクター作成レギュレーション
## ◆ キャラクター作成の基本ルール
- キャラクターの能力値決定ダイスは、通常のルールブック作成方法の通り、全体を 3 回振った中から好きな 1 回を選ぶ方法で決定するが、以下の追加ルールを適用する。
  - 能力値は、それぞれ個別に 3 回までは振り直しても良い。
    - ただし、振り直した場合、その能力値決定ダイスの最終結果を採用する。
  - 能力値の決定ダイスは、同じダイス数の能力値決定ダイス間であれば、回数の制限なく、ダイス結果の入れ替えを自由に許可する。
    - 例1
      - A：2D の結果「6」が出た。B：2D の結果「8」が出た場合、A を「8」、B を「6」に入れ替えても良い。
    - 例2
      - A：2D の結果「4」が出た。B：2D + 3 の結果「7 + 3 = 10」が出た場合、A を「7」、B を「4 + 3 = 7」に入れ替えても良い。
  - ただし、異なるダイス数の能力値決定ダイス間での入れ替えは許可しない。
    - 例
      - A：2D の結果「6」が出た。B：1D + 6 の結果「3 + 6 = 9」が出た場合、A と B のダイス結果を入れ替えてはいけない。
